mARker: Hybrid-Interfaced Spatial Sketching via iPad AR, Apple Pencil, and an Instantly Crafted Tracker

Mobile AR promises wider access to spatial sketching, but existing designs either pack interactions into dense on-screen controls and constant device motion, or impose additional hardware barriers. mARker explores a hybrid iPad AR + Apple Pencil interface that uses a low-cost, self-fabricated cube for stylus tracking and gesture-based mode switching.

Development

mARker's AR app is built under iPadOS 17.5 and supports two interfaces: a spatial interface for mid-air sketching via stylus tracking, and an on-screen interface for proxy plane sketching, entity transformation, along with auxiliary controls such as color selection.




The development of mARker benefits greatly from the new APIs introduced at WWDC 24 for Apple Pencil Pro. We implemented two sketching modes using these APIs: freehand and ruler. double-tap serves as a global toggle between these modes, while squeeze and barrel-roll are assigned to interface-specific sketching functions.

Interactions

With mARker, user can directly sketch mid-air 3D content, or use traditional way to sketch projected 3D content and manipulate the stroke entities. Practical scenarios are introduced, including in-situ analysis diagram annotation on real object, and dimensioning an object by physically tracing it and evaluating the dimension.
